# HDI.FinancialRiskEngine

This project is a financial risk analysis platform developed as a case study.

## 🧠 Features
- Multi-tenant architecture
- Agreement & partner management
- Business topic intake via API
- Keyword-based risk analysis engine
- RESTful API
- Clean Architecture

## 🏗️ Tech Stack
- .NET Core
- Entity Framework Core
- MSSQL
- FluentValidation
- Clean Architecture

## ⚙️ Modules
- Agreements
- Business Partners
- Business Topics
- Risk Analysis Engine

## 🚀 How It Works
1. Business partners send topics via API
2. System stores incoming data
3. Keywords are matched
4. Risk score is calculated
5. Financial risk output is generated

## 📌 Architecture
- Domain
- Application
- Infrastructure
- WebApi
- WebUI
